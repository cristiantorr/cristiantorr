# 👋 Hola, soy Cristian Torres - WEB DEVELOPER

Soy Tecnólogo en Análisis y Desarrollo de Sistemas de Información con más de 10 años de experiencia en el desarrollo de aplicaciones web, plataformas digitales y soluciones empresariales.

A lo largo de mi trayectoria he participado en el análisis, diseño, desarrollo, implementación y mantenimiento de aplicaciones web, especializándome en PHP, WordPress y React. Cuento con experiencia en el desarrollo de plugins, funcionalidades personalizadas, integración de APIs REST y optimización de aplicaciones enfocadas en rendimiento, escalabilidad, seguridad y experiencia de usuario.

Como Full Stack Developer, he participado en el desarrollo de aplicaciones utilizando React, realizando maquetación de interfaces modernas con Tailwind CSS y Bootstrap, además de integrar aplicaciones con Salesforce CRM mediante el consumo de APIs REST. También he adquirido experiencia práctica en AWS (EC2) realizando despliegues de aplicaciones, configuración básica de servidores y tareas de administración y mantenimiento de infraestructura.

Adicionalmente, he trabajado como Freelance Web Developer, desarrollando soluciones web personalizadas para diferentes clientes, adaptándome a distintos sectores y necesidades de negocio, desde sitios corporativos hasta aplicaciones con funcionalidades específicas.

Me considero una persona apasionada por la tecnología, el aprendizaje continuo y la resolución de problemas. Disfruto trabajar en equipos multidisciplinarios, compartir conocimiento y participar en todo el ciclo de vida del desarrollo de software, aplicando buenas prácticas de programación, código limpio, control de versiones con Git y metodologías orientadas a la entrega de soluciones escalables y mantenibles.
